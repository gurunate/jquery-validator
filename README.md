# jQuery Validator plugin #

Quick and easy jQuery inline form validation.

## Getting Started ##

* Install [NodeJS](http://nodejs.org/ "node.js")
* Install [Grunt](http://gruntjs.com/ "Grunt")
* Clone this project
* Run ``npm install``
* Run ``grunt`` to build

## Development ##

* Run ``node app.js`` to view on Express server
